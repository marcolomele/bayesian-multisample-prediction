Creating an Enviorment:

conda installl <package>

Navigating in Linux:
mkdir -p ~/IMA_Project/{scripts,data,outputs,logs}
‘cd’ —> change directory
‘cd -‘ —> previous directory
‘ls’ —> see whats in your current directory
‘chmod +x <file.type>’ —> sets file permission to executable
source activate <enviorment_name>	—> enter enviorment
conda deactivate —> exit enviorment

SLURM:
nano -u ~/<ENTER_FILE_PATH>		—> this will open SLURM
